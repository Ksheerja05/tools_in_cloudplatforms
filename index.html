Data Visualization & Analytical Tools

AWS

Amazon QuickSight:

  • Managed business intelligence (BI) service.
  • Allows creation of interactive dashboards and reports.
  • Seamless integration with various AWS data sources.

    • AWS

    Amazon Redshift:

  • Fully managed data warehouse service.
  • Ideal for analytical queries on large datasets.
  • Integrates with popular BI and data visualization tools.

    • AWS

    AWS Glue:

  • ETL (Extract, Transform, Load) service.
  • Automatically discovers, catalogs, and moves data.
  • Prepares data for analytics with a serverless, fully managed extract, transform, and load (ETL) service.

    • GCP

    Google Data Studio: Free tool for customizable and shareable data dashboards.

    BigQuery: Serverless, scalable data warehouse supporting real-time analytics and integration with machine learning services.

    Cloud Dataflow: Managed service for stream and batch processing.

    Azure

    Power BI: Data visualization and business analytics tool for interactive reports. It integrates with a wide range of data sources, both on-premises and in the cloud.

    Azure Synapse Analytics: Unified analytics service integrating big data and data warehousing. Supports both on-demand and provisioned resources for analytics.

    Azure Stream Analytics: Real-time analytics service for processing streaming data, enabling real-time insights and actions based on the data.
