Data Visualization & Analytical Tools

AWS

  • Amazon QuickSight: Managed BI service for interactive dashboards and reports.
  • Amazon Redshift: Fully managed data warehouse for analytical queries on large datasets.
  • AWS Glue: ETL service for data discovery, cataloging, and preparation.

GCP

  • Google Data Studio: Free tool for customizable and shareable data dashboards.
  • BigQuery: Serverless, scalable data warehouse supporting real-time analytics.
  • Cloud Dataflow: Managed service for stream and batch processing.

Azure

  • Power BI: Data visualization and business analytics tool for interactive reports.
  • Azure Synapse Analytics: Unified analytics service integrating big data and data warehousing.
  • Azure Stream Analytics: Real-time analytics service for processing streaming data.
